1. Create 'bower.json', 'package.json', '.bowerrc', 'gulpfile.js'
	cmd: 'bower init', 'npm init'
2. Frameworks and libs - "bower install -S". Other plagins - "npm instal -D"
	Плагины
gulp-wiredep - автоматическая запись библиотке и фреймворков в .html.
gulp-useref - автоматическая сборка проекта в папке dist.
gulp-if - фильтрует подключенные к .html файлы.
gulp-uglify - минифицирует javascript.
gulp-minify-css - минифицирует css.

"npm start" - http://localhost:8000/app/index.html
